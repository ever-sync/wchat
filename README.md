# DistribuiBot Pro

Painel operacional com clientes, campanhas, inbox e follow-up de WhatsApp usando Supabase + UAZAPI.

## Rodar localmente

```sh
npm install
npm run dev
```

Preencha [`.env.example`](/C:/Users/rapha/OneDrive/Documentos/GitHub/distribui-bot-pro/.env.example) antes de iniciar a integracao real com Supabase.

## Deploy do WhatsApp/UAZAPI

O passo a passo de banco, functions, secrets e automacao ficou em [SUPABASE_UAZAPI_DEPLOY.md](/C:/Users/rapha/OneDrive/Documentos/GitHub/distribui-bot-pro/docs/SUPABASE_UAZAPI_DEPLOY.md).

## Stack

- Vite
- React
- TypeScript
- Tailwind
- Supabase
